# PongOnRoids

## Install guide

### Windows:
```bash
git clone --recurse-submodules https://github.com/MV487/PongOnRoids.git
cd vcpkg\bootstrap-vcpkg.bat
vcpkg.exe install raylib:x64-windows
vcpkg.exe integrate install
```
